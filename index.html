My Projects

Project 1:

Car Game

"The car game was developed using the Python programming language. In this project, we utilized various classes and modules of Python. By calling different methods, we designed our game. As a result, a 2D game was created in which the user can control their car using the left and right arrow keys to shift the car between lanes on the road and avoid colliding with enemy cars coming from the opposite side of the lane. The car's speed also increases after scoring every 5 points. When the user loses, the game will ask the user whether they want to play again or not."

Project 2:

3D Dot Product of Two Vectors

"This project was developed using the Python programming language to calculate the dot product of two vectors and plot it in a 3D plane with x, y, and z-axes. The project was built using the Matplotlib library and Tkinter. It features two input boxes where users can enter the values of two three-dimensional vectors and then click the 'Dot Product' button to calculate the result. Additionally, there is a 'Plot' button that displays a 3D graph of the final product."
